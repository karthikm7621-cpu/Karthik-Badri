# Support

Thank you for using this project.

## Community Support

For general questions, usage help, and troubleshooting, please open a GitLab issue:

[Project Issues](https://code.swecha.org/Badrinath/team-atk/-/issues)

Before opening an issue, please check:

- Existing open and closed issues
- The `README.md`
- The `USER_MANUAL.md`
- The `CHANGELOG.md`

## Security Issues
